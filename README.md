# Suhaffinity — Software Development Company Website

Suhaffinity is a modern, responsive company website built using React and Vite.
The website represents a software development company and introduces its services,
product concept, mission, vision, and team members.

Slogan: Id Fieri Facimus (We Make It Happen)

---

## Project Overview

This project was developed as part of CAT201 – Integrated Software Development Workshop
(Semester 1, 2025/2026). The objective is to apply modern web development tools and GUI
design principles to create a professional company homepage.

---

## How to Run the Project Locally

1. Clone the repository
   git clone https://github.com/suhaff/Suhaffinity.git

2. Navigate to the project directory
   cd suhaffinity-website

3. Install dependencies
   npm install

4. Start the development server
   npm run dev

5. Open the application in a browser
   http://localhost:5173
